Flat Earth CSS

A super tiny CSS framework

By Rushil Saraogi

Installation

Using NPM

npm install flat-earth-css

Usage

As an ES6 import. Place this line in the root of your application.

import "flat-earth-css";

Buttons

// Default button

<button class="btn">DEFAULT BUTTON</button>

// Outlined button

<button class="btn-outline">OUTLINED BUTTON</button>

// Clear button

<button class="btn-clear">CLEAR BUTTON</button>
